# Contributing

Community contributions to this project are welcome! Please review the `CONTRIBUTING.md` [guide](https://github.com/cadeluca/hextraction-catalog/blob/main/CONTRIBUTING.md) in the GitHub repository.

Here are some great resources pertaining to writing markdown, or development in the context of Nextra, which is built on top of Next.js, a JavaScript framework:
- [Javascript Guide](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)
- [Markdown Guide](https://www.markdownguide.org/getting-started/)
- [MDX](https://mdxjs.com/)
- [Next.js](https://nextjs.org/docs)
- [Nextra](https://nextra.site/docs)
